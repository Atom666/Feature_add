Теперь задача такая - внедрить аутентификацию. Главные особенности:
1. Без входа человек не видит никаких тасков
2. При входе человек может видеть только те таски, которые были созданы им же т.е. его пользователем. Для этого я добавил в бд у задач новую колонку - user_id. Она будет отвечать за изоляцию тасков между пользователями т.е. пользователь будет видеть только те таски, у которых user_id совпадает с user_id его пользователя.
3. Добавь кнопку logout, чтобы пользователь мог выходить из своего аккаунта
